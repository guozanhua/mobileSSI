EHealth
  EHealth  SENSOR
 
           Connects glove device via serial com port.
 
             +  port   UINT   1   '0'   LOCK   number of com port 
             +  sr   UINT   1   '100'   LOCK   sampling rates (hz) 
 
           Available channels:
 
             +  ecg   Provides measure of electrocardiogram (ECG). 
             +  gsr   Provides measure of skin conductance, also known as galvanic skin response (GSR). 
             +  air   Provides measure of airflow. 
             +  tmp   Provides measure of temperature. 
             +  bpm   Provides measure pulse (bpm). 
             +  oxy   Provides measure of oxygen saturation.
